# ComfyUI Prompt Library

A collection of ComfyUI nodes for efficient prompt management and organization. Handle multi-line prompts with positive/negative separation, create prompt libraries, and extend prompts dynamically.


__Prompt Library:__

copy the models folder from the repository to your ComfyUI to get the demo project files.

```
This is the Syntax of the project files. 
### For a prompt name in the same line.
--- or ---- or ----- to seperate positive and negative prompt
Empty line to seperate different prompts.
```
Demo Projects
```
### Forest Meadow (Just a name for your prompt)
```
